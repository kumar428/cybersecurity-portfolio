## 2. Key Concepts Understood

### 5 Phases of Ethical Hacking
1. **Reconnaissance** — Gather information about the target (IP, domain, infrastructure)
2. **Scanning** — Identify open ports, services, and potential entry points using tools like Nmap
3. **Gaining Access** — Exploit identified vulnerabilities to enter the system
4. **Maintaining Access** — Establish persistence within the system for continued testing
5. **Covering Tracks** — Remove evidence of activity to simulate real attacker behavior

### Vulnerabilities Discovered in Lab
- **Information Disclosure** — Account data visible without authentication
- **Broken Access Control** — Admin panel accessible without login (/bank-transfer)
- **Business Logic Flaw** — Unrestricted fund transfer with no validation
